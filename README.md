## Diabetes Risk Prediction with Machine Learning

### Objective
Build a machine learning model to predict diabetes risk using patient-level clinical features.

### Methods
- Exploratory Data Analysis
- Clinically informed Feature Engineering
- Logistic Regression (baseline model)
- Random Forest (nonlinear model)

### Evaluation Metric
ROC-AUC

### Results
Logistic Regression ROC-AUC: **0.84**

Random Forest ROC-AUC: **0.81**

### Key Predictors
Top predictors identified by the models include:

- Glucose
- Metabolic risk score
- BMI
- Diabetes pedigree function
- Age

These results are consistent with known clinical risk factors for diabetes.
